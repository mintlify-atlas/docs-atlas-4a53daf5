### Controllers Organization

Safaris Web organizes controllers by feature:

- **HomeController.php** (`~/workspace/source/app/Http/Controllers/HomeController.php`)
  - Handles static pages (about, careers, FAQs)
  - Homepage with featured tours
  - Search functionality

- **ToursController.php** 
  - CRUD operations for tour packages
  - Publishing/unpublishing tours
  - Admin dashboard views

- **BookingController.php**
  - Customer booking management
  - User profile/booking history
  - Booking confirmation workflow

- **Resource Controllers** (Products, Services, Events, Accommodations)
  - Follow RESTful conventions
  - Category filtering
  - Slug-based URLs

### Layout System

Safaris Web uses a master layout pattern (`~/workspace/source/resources/views/layout/index.blade.php:1`):

```php
<!DOCTYPE html>
<html lang="{{ str_replace('_', '-', app()->getLocale()) }}">
<head>
    <!-- Meta tags, CSS, fonts -->
</head>
<body>
    <!-- Navigation bar -->
    <nav class="navbar navbar-expand-lg">...</nav>
    
    <!-- Main content area -->
    @yield('content')
    
    <!-- Footer -->
    <div class="footer">...</div>
</body>
</html>
```

Child views extend this layout:

```php
@extends('layout.index')

@section('content')
    <!-- Page-specific content -->
@endsection
```

### Route Organization

The `web.php` file (`~/workspace/source/routes/web.php:1`) organizes routes by feature:

```php
// Static pages
Route::get('/', [HomeController::class, 'index']);
Route::get('/about', [HomeController::class, 'about']);

// Tours (with auth middleware)
Route::get('/addtour', [ToursController::class, 'index'])
    ->middleware(['auth', 'verified']);

// Bookings (with auth + email verification)
Route::get('/bookings', [BookingController::class, 'index'])
    ->middleware(['auth', 'verified']);

// Resource routes
Route::resource('enquiries', EnquiryController::class);
Route::resource('highlights', HighlightController::class);
```

## MVC Architecture

### Model-View-Controller Flow

1. **Request** → Routes (`routes/web.php`)
2. **Route** → Controller (`app/Http/Controllers/`)
3. **Controller** → Model (`app/Models/`)
4. **Model** ↔ Database
5. **Controller** → View (`resources/views/`)
6. **Response** ← Compiled HTML

### Example: Tour Listing Flow

```php
// 1. Route (routes/web.php:67)
Route::get('/safaris', [ToursController::class, 'safaris']);

// 2. Controller (app/Http/Controllers/ToursController.php:21-27)
public function safaris() {
    $categories = Category::all();
    $blogs = Blog::latest()->limit(2)->get();
    $safaris = Tours::inRandomOrder()->paginate(20);
    return view('safaris.index', compact('categories', 'blogs', 'safaris'));
}

// 3. Model (app/Models/Tours.php)
class Tours extends Model {
    protected $fillable = ['hotel', 'transport', 'image', ...];
}

// 4. View (resources/views/safaris/index.blade.php)
@foreach($safaris as $safari)
    <div class="tour-card">{{ $safari->hotel }}</div>
@endforeach
```

## Laravel Conventions

### Naming Conventions

| Type | Convention | Example |
|------|------------|----------|
| Model | Singular, PascalCase | `Tour`, `Booking` |
| Controller | PascalCase + "Controller" | `ToursController` |
| Migration | snake_case + description | `create_tours_table` |
| View | snake_case.blade.php | `create.blade.php` |
| Route | kebab-case | `/tour-bookings` |
| Table | Plural, snake_case | `tour_bookings` |

### Controller Methods (RESTful)

- `index()` - List resources
- `create()` - Show creation form
- `store()` - Save new resource
- `show($id)` - Display specific resource
- `edit($id)` - Show edit form
- `update($id)` - Update resource
- `destroy($id)` - Delete resource

### Custom Methods

Safaris Web adds custom methods:
- `publish($id)` - Toggle published status
- `confirm($id)` - Confirm booking
- `safaris()` - Public listing page

## Autoloading

Composer PSR-4 autoloading configuration (`composer.json:25-30`):

```json
"autoload": {
    "psr-4": {
        "App\\": "app/",
        "Database\\Factories\\": "database/factories/",
        "Database\\Seeders\\": "database/seeders/"
    }
}
```

After adding new classes:
```bash
composer dump-autoload
```

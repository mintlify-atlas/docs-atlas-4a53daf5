## Overview

Safaris Web uses Laravel's Blade templating engine for all views. Templates are organized in `~/workspace/source/resources/views/` with a master layout system for consistent design.

## Directory Structure

```
resources/views/
├── layout/                 # Master layouts
│   ├── index.blade.php     # Main public layout
│   ├── admin.blade.php     # Admin dashboard layout
│   ├── dash.blade.php      # Alternative dashboard
│   └── new.blade.php       # Alternative layout
├── auth/                   # Authentication views
│   ├── login.blade.php
│   ├── register.blade.php
│   └── verify.blade.php
├── tours/                  # Tour management
│   ├── index.blade.php
│   ├── create.blade.php
│   ├── edit.blade.php
│   └── show.blade.php
├── bookings/               # Booking views
├── blogs/                  # Blog views
├── products/               # Product catalog
├── services/               # Service views
├── events/                 # Event views
├── accomodations/          # Accommodation views
├── admin/                  # Admin dashboard
└── home.blade.php          # Homepage
```

## Common Blade Directives

### Control Structures

<CodeGroup>

```php If Statements
@if($tours->count() > 0)
    <p>We have {{ $tours->count() }} tours available.</p>
@elseif($tours->count() == 0)
    <p>No tours available at the moment.</p>
@else
    <p>Something went wrong.</p>
@endif
```

```php Unless
@unless(Auth::check())
    <p>You are not signed in.</p>
@endunless
```

```php Authentication
@auth
    <p>Welcome, {{ Auth::user()->name }}!</p>
@endauth

@guest
    <p>Please log in.</p>
@endguest
```

```php Isset/Empty
@isset($tours)
    <p>Tours variable is defined.</p>
@endisset

@empty($tours)
    <p>No tours found.</p>
@endempty
```

</CodeGroup>

### Loops

<CodeGroup>

```php Foreach Loop
@foreach($tours as $tour)
    <div class="tour-card">
        <h3>{{ $tour->hotel }}</h3>
        <p>{{ $tour->location }}</p>
    </div>
@endforeach
```

```php For Loop
@for($i = 1; $i <= 5; $i++)
    <i class="bi bi-star"></i>
@endfor
```

```php Forelse (with empty state)
@forelse($tours as $tour)
    <div>{{ $tour->hotel }}</div>
@empty
    <p>No tours available.</p>
@endforelse
```

```php Loop Variable
@foreach($tours as $tour)
    @if($loop->first)
        <p>This is the first iteration.</p>
    @endif
    
    <p>Tour #{{ $loop->index }}: {{ $tour->hotel }}</p>
    
    @if($loop->last)
        <p>This is the last iteration.</p>
    @endif
@endforeach
```

</CodeGroup>

### Forms

#### CSRF Protection

All POST forms must include CSRF token:

```php
<form action="{{ route('tour.store') }}" method="POST">
    @csrf
    <!-- form fields -->
</form>
```

#### Method Spoofing

For PUT, PATCH, DELETE requests:

```php
<form action="{{ route('tour.update', $tour->id) }}" method="POST">
    @csrf
    @method('PUT')
    <!-- form fields -->
</form>
```

#### Validation Errors

```php
@if($errors->any())
<div class="alert alert-danger">
    <ul>
        @foreach($errors->all() as $error)
        <li>{{ $error }}</li>
        @endforeach
    </ul>
</div>
@endif

<!-- Individual field error -->
<input type="text" name="hotel" class="form-control @error('hotel') is-invalid @enderror" value="{{ old('hotel') }}">
@error('hotel')
    <span class="invalid-feedback">{{ $message }}</span>
@enderror
```

#### Flash Messages

```php
@if(session('success'))
<div class="alert alert-success">
    {{ session('success') }}
</div>
@endif

@if(session('error'))
<div class="alert alert-danger">
    {{ session('error') }}
</div>
@endif
```

### Data Display

<CodeGroup>

```php Escaped Output
<!-- HTML-escaped (safe) -->
<p>{{ $tour->hotel }}</p>
<p>{{ $user->name }}</p>
```

```php Raw Output
<!-- Unescaped (use with caution) -->
<div>{!! $blog->content !!}</div>
```

```php With Default
<!-- Show default if variable is empty -->
<p>{{ $tour->additional_info ?? 'No additional information' }}</p>
```

</CodeGroup>

## Best Practices

### 1. Always Escape User Input

```php
<!-- Good -->
<p>{{ $user->name }}</p>

<!-- Bad (XSS vulnerability) -->
<p>{!! $user->name !!}</p>
```

### 2. Use Named Routes

```php
<!-- Good -->
<a href="{{ route('tour.show', $tour->id) }}">View Tour</a>

<!-- Avoid -->
<a href="/tours/{{ $tour->id }}">View Tour</a>
```

### 3. Keep Logic in Controllers

```php
<!-- Good: Simple display -->
<p>{{ $tour->formatted_price }}</p>

<!-- Bad: Complex logic in view -->
<p>{{ '$' . number_format($tour->per_person_sharing, 2) }}</p>
```

### 4. Use Blade Components for Reusable UI

Create components for repeated HTML structures.

### 5. Optimize Images

Use responsive images:

```php
<img src="{{ asset('storage/' . $tour->image) }}" 
     class="img-fluid" 
     alt="{{ $tour->hotel }}" 
     loading="lazy">
```

## JavaScript Integration

Example from layout (`index.blade.php:400-421`):

```php
<script>
// Back to top button
myButton = document.getElementById("myBtn");

window.onscroll = function () {
    scrollFunction();
};

function scrollFunction() {
    if (document.body.scrollTop > 20 || document.documentElement.scrollTop > 20) {
        myButton.style.display = "block";
    } else {
        myButton.style.display = "none";
    }
}

function topFunction() {
    document.body.scrollTop = 0;
    document.documentElement.scrollTop = 0;
}
</script>
```

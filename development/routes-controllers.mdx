## Overview

Safaris Web defines all web routes in `~/workspace/source/routes/web.php` and organizes functionality through dedicated controllers in `~/workspace/source/app/Http/Controllers/`.

### Resource Routes

Laravel's resource routes provide RESTful CRUD operations:

```php routes/web.php:53,117
Route::resource('enquiries', EnquiryController::class);

```

This creates the following routes:

| HTTP Method | URI | Action | Route Name |
|-------------|-----|--------|------------|
| GET | /enquiries | index | enquiries.index |
| GET | /enquiries/create | create | enquiries.create |
| POST | /enquiries | store | enquiries.store |
| GET | /enquiries/{id} | show | enquiries.show |
| GET | /enquiries/{id}/edit | edit | enquiries.edit |
| PUT/PATCH | /enquiries/{id} | update | enquiries.update |
| DELETE | /enquiries/{id} | destroy | enquiries.destroy |

    $request->validate([
        'hotel' => 'required',
        'transport' => 'required',
        'image' => 'required|mimes:jpg,jpeg,svg,png,webp',
        // ... more validation
    ]);

    $path = $request->image->store('images', 'public');
    // ... handle multiple images
    $tours->hotel = $request->hotel;

```php Custom Methods
// Toggle publish status
public function publish($id)
{
    $tour = Tours::find($id);
    
    if ($tour->published == 0) {
        $tour->published = 1;
        $tour->update();
    } else {
        $tour->published = 0;
        $tour->update();
    }
    
    return redirect()->route('tours');
}
```

### BookingController

**Location**: `~/workspace/source/app/Http/Controllers/BookingController.php`

**Purpose**: Customer booking management and user profiles.

#### Key Methods

```php
// User's booking history
public function profile()
{
    $bookings = Booking::where('user_id', Auth::user()->id)
        ->latest()
        ->paginate(10);
    return view('users.profile', compact('bookings'))
        ->with('i', (request()->input('page', 1) - 1) * 5);
}

// Booking form
public function create($destination)
{
    $tours = Tours::latest()->limit(3)->get();
    $categories = Category::all();
    $blogs = Blog::all();
    
    return view('bookings.create', [
        'categories' => $categories,
        'tours' => $tours,
        'blogs' => $blogs,
        'destination' => $destination
    ]);
}

// Store booking
public function store(Request $request)
{
    $request->validate([
        'first_name' => 'required',
        'last_name' => 'required',
        'phone_number' => 'required',
        'arrival_date' => 'required',
        'adults' => 'required',
        'kids' => 'required',
        'destination' => 'required',
        'g-recaptcha-response' => 'required|captcha',
        'transaction_code' => 'required'
    ]);
    
    // Email validation
    if($request->email1 == $request->email2) {
        $email = $request->email1;
    } else {
        return redirect()->route('bookings.create')
            ->with('failure', 'please provide matching emails');
    }
    
    $booking = new Booking;
    // ... set properties
    $booking->user_id = Auth::user()->id;
    $booking->save();
    
    return redirect()->route('profile')
        ->with('success', 'Booking has been successfull, We will contact you within 24hrs!');
}

// Confirm booking (admin)
public function confirm($id)
{
    $booking = Booking::find($id);
    $booking->status = ($booking->status == 0) ? 1 : 0;
    $booking->update();
    
    return redirect()->route('bookings');
}
```

## Authentication Routes

```php routes/web.php:120
Auth::routes(['verify' => true]);
```

This generates:
- `GET /login` - Login form
- `POST /login` - Process login
- `POST /logout` - Logout
- `GET /register` - Registration form
- `POST /register` - Process registration
- `GET /password/reset` - Password reset request
- `POST /password/email` - Send reset link
- `GET /password/reset/{token}` - Reset password form
- `POST /password/reset` - Process password reset

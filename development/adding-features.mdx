## Overview

This guide demonstrates how to add new features to Safaris Web by following the established patterns in the codebase.

## Feature Development Workflow

When adding a new feature, follow this sequence:

1. **Database**: Create migration
2. **Model**: Create Eloquent model
3. **Controller**: Create controller with methods
4. **Routes**: Define routes
5. **Views**: Create Blade templates
6. **Test**: Verify functionality

## Example: Adding a "Testimonials" Feature

Let's walk through adding a complete testimonials management system.

### Step 1: Create Migration

```bash
php artisan make:migration create_testimonials_table
```

Edit the migration file:

```php database/migrations/xxxx_create_testimonials_table.php
use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Support\Facades\Schema;

return new class extends Migration
{
    public function up()
    {
        Schema::create('testimonials', function (Blueprint $table) {
            $table->id();
            $table->string('name');
            $table->string('country');
            $table->string('image')->nullable();
            $table->text('content');
            $table->integer('rating')->default(5);
            $table->integer('user_id');
            $table->integer('published')->default(0);
            $table->timestamps();
        });
    }

    public function down()
    {
        Schema::dropIfExists('testimonials');
    }
};
```

Run the migration:

```bash
php artisan migrate
```

### Step 2: Create Model

```bash
php artisan make:model Testimonial
```

Edit the model following existing patterns:

```php app/Models/Testimonial.php
namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;

class Testimonial extends Model
{
    use HasFactory;

    protected $fillable = [
        'name',
        'country',
        'image',
        'content',
        'rating'
    ];

    // Relationship to user (creator)
    public function creator()
    {
        return $this->belongsTo(User::class, 'user_id');
    }
}
```

### Step 3: Create Controller

```bash
php artisan make:controller TestimonialController --resource
```

Implement controller methods following the ToursController pattern:

```php app/Http/Controllers/TestimonialController.php
namespace App\Http\Controllers;

use App\Models\Testimonial;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Auth;

class TestimonialController extends Controller
{
    // Public listing (published only)
    public function index()
    {
        $testimonials = Testimonial::where('published', 1)
            ->latest()
            ->paginate(12);
        return view('testimonials.index', compact('testimonials'));
    }

    // Admin listing (all testimonials)
    public function all()
    {
        $testimonials = Testimonial::latest()->paginate(20);
        return view('testimonials.admin', compact('testimonials'))
            ->with('i', (request()->input('page', 1) - 1) * 5);
    }

    // Show create form
    public function create()
    {
        return view('testimonials.create');
    }

    // Store new testimonial
    public function store(Request $request)
    {
        $request->validate([
            'name' => 'required|string|max:255',
            'country' => 'required|string|max:255',
            'content' => 'required',
            'rating' => 'required|integer|min:1|max:5',
            'image' => 'nullable|mimes:jpg,jpeg,png,webp|max:2048',
        ]);

        $testimonial = new Testimonial();
        $testimonial->name = $request->name;
        $testimonial->country = $request->country;
        $testimonial->content = $request->content;
        $testimonial->rating = $request->rating;
        $testimonial->user_id = Auth::id();

        // Handle image upload
        if ($request->hasFile('image')) {
            $path = $request->image->store('images', 'public');
            $testimonial->image = $path;
        }

        $testimonial->save();

        return redirect()->route('testimonials.admin')
            ->with('success', 'Testimonial added successfully!');
    }

    // Show single testimonial
    public function show($id)
    {
        $testimonial = Testimonial::findOrFail($id);
        return view('testimonials.show', compact('testimonial'));
    }

    // Show edit form
    public function edit($id)
    {
        $testimonial = Testimonial::findOrFail($id);
        return view('testimonials.edit', compact('testimonial'));
    }

    // Update testimonial
    public function update(Request $request, $id)
    {
        $request->validate([
            'name' => 'required|string|max:255',
            'country' => 'required|string|max:255',
            'content' => 'required',
            'rating' => 'required|integer|min:1|max:5',
            'image' => 'nullable|mimes:jpg,jpeg,png,webp|max:2048',
        ]);

        $testimonial = Testimonial::findOrFail($id);
        $testimonial->name = $request->name;
        $testimonial->country = $request->country;
        $testimonial->content = $request->content;
        $testimonial->rating = $request->rating;

        // Handle image upload
        if ($request->hasFile('image')) {
            $path = $request->image->store('images', 'public');
            $testimonial->image = $path;
        }

        $testimonial->update();

        return redirect()->route('testimonials.admin')
            ->with('success', 'Testimonial updated successfully!');
    }

    // Delete testimonial
    public function destroy($id)
    {
        $testimonial = Testimonial::findOrFail($id);
        $testimonial->delete();

        return redirect()->route('testimonials.admin')
            ->with('success', 'Testimonial deleted successfully!');
    }

    // Toggle publish status
    public function publish($id)
    {
        $testimonial = Testimonial::findOrFail($id);
        $testimonial->published = ($testimonial->published == 0) ? 1 : 0;
        $testimonial->update();

        return redirect()->route('testimonials.admin')
            ->with('success', 'Testimonial status updated!');
    }
}
```

### Step 4: Define Routes

Add routes to `routes/web.php`:

```php routes/web.php
// Public testimonials
Route::get('/testimonials', [TestimonialController::class, 'index'])
    ->name('testimonials');

Route::get('/testimonial/{id}', [TestimonialController::class, 'show'])
    ->name('testimonial.show');

// Admin testimonials (protected)
Route::get('/all-testimonials', [TestimonialController::class, 'all'])
    ->name('testimonials.admin')
    ->middleware(['auth', 'verified']);

Route::get('/testimonial/create', [TestimonialController::class, 'create'])
    ->name('testimonial.create')
    ->middleware(['auth', 'verified']);

Route::post('/testimonial/store', [TestimonialController::class, 'store'])
    ->name('testimonial.store')
    ->middleware(['auth', 'verified']);

Route::get('/testimonial/edit/{id}', [TestimonialController::class, 'edit'])
    ->name('testimonial.edit')
    ->middleware(['auth', 'verified']);

Route::put('/testimonial/update/{id}', [TestimonialController::class, 'update'])
    ->name('testimonial.update')
    ->middleware(['auth', 'verified']);

Route::delete('/testimonial/delete/{id}', [TestimonialController::class, 'destroy'])
    ->name('testimonial.delete')
    ->middleware(['auth', 'verified']);

Route::put('/testimonial/publish/{id}', [TestimonialController::class, 'publish'])
    ->name('testimonial.publish')
    ->middleware(['auth', 'verified']);
```

## Following Existing Patterns

### Controller Patterns

Safaris Web controllers follow these conventions:

1. **Separate public and admin methods**
   - `index()` for public listing
   - `all()` for admin listing

2. **Consistent validation**
   ```php
   $request->validate([
       'field' => 'required|type|max:length',
   ]);
   ```

3. **Flash messages for feedback**
   ```php
   return redirect()->route('route.name')
       ->with('success', 'Action completed!');
   ```

4. **Publish toggle pattern**
   ```php
   public function publish($id) {
       $model = Model::find($id);
       $model->published = ($model->published == 0) ? 1 : 0;
       $model->update();
       return redirect()->back();
   }
   ```

### Image Upload Pattern

From ToursController.php:80-86:

```php
if ($request->hasFile('image')) {
    $path = $request->image->store('images', 'public');
    $model->image = $path;
}
```

### Model Patterns

1. **Always use HasFactory trait**
2. **Define fillable fields**
3. **Add relationships**
4. **Follow singular naming** (Testimonial, not Testimonials)

### Route Patterns

1. **Named routes** for easier reference
2. **Middleware for protection**
3. **RESTful naming conventions**
4. **Group related routes**

### View Patterns

1. **Extend layouts**: `@extends('layout.index')`
2. **Use sections**: `@section('content')`
3. **Bootstrap 5 classes** for styling
4. **Blade directives** for logic
5. **Asset helper** for files: `asset('storage/' . $path)`

## Adding API Endpoints

If you need API access, add routes to `routes/api.php`:

```php routes/api.php
use App\Http\Controllers\TestimonialController;

Route::middleware('auth:sanctum')->group(function () {
    Route::get('/testimonials', [TestimonialController::class, 'apiIndex']);
    Route::get('/testimonials/{id}', [TestimonialController::class, 'apiShow']);
});
```

Add API methods to controller:

```php
public function apiIndex()
{
    return response()->json([
        'data' => Testimonial::where('published', 1)->get()
    ]);
}
```
